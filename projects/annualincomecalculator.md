For my ICS 212 class, we were tasked with creating an annual income calculator that would be able to calculate the user's total owed tax, both state and federal, witin a given year. This was one of the first programs I ever created using C/C++. It's simplistic of course, but this program taught me a lot about working with C/C++, because I had only ever used Java before this class. 

<img class="img-fluid" src="/img/forwhile.png">

I pretty much had to learn everything from scratch. The commands, while relatively the same, felt foreign in this new program. There were different commands that I had to learn, that I never used with Java. Eventually, I got used to the different commands and managed to complete a working program. One of the main problems I faced when creating this program was trying to understand the different uses of For and While loops. Having loops within loops was even more confusing. However, with some help from my professor, the program ran smoothly
