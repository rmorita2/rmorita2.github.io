For my ICS 211 class we were tasked with creating a fully interactive Linked List. The project would open up to a menu where users can input any number between 1 and 10, and based on what the user inputed, the program would then insert, delete, find, and many other possible functions, of any given integer that the user inputs, into a fully customizable linked list. For example, a user can input the integer 7 at the beginning, or head, of the list. Add an 8 to the tail, or end of the linked list, and delete 7 from the list. Users could also print the full linked list to view, or even find out how many instances there are of a certain integer.

<img class="img-fluid" src="/img/ll2.png">

This was actually a really fun project to do. I enjoyed learning about how Linked Lists work and how to create an interactive menue interface. While most of the commands were easy to implement, others required a bit more coding to get working properly. For example, one of the options is to insert an integer at any point in the linked list. To do this, I had to create a While loop that would find if an availible position is within the linked list, it would then insert the integer if there was an availible position, or report an error message to the user. For example, if a user wanted to insert a 7 at position 5 when there are only 3 integers in the linked list, an error message would appear.
